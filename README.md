# WpfAdmin

简体中文说明文档（简要）

## 项目简介
`WpfAdmin` 是一个基于ABP vNext的WPF桌面管理后台示例/模版，包含典型的 MVVM 结构，用于演示桌面端管理系统的架构与实现思路。

## 主要特性
- 基于 WPF 的桌面 UI（MVVM 模式）
- 模块化的视图与视图模型组织
- 基于ABP vNext

## 目录结构
- `Admin.Desktop` — WPF 客户端
- `Admin.HttpApi.Host` — 接口服务端
